NBA Viz

Problem Statement

In basketball world, one of the most prevalent question people ask is about the relationship between player’s size and performance. In order to find out the correlation, we conducted this explorative project to learn and visualize the trend of NBA players’ average height and weight, as well as their performance by using dataset from 1950 to 2018. The goal of the project is to find out how NBA players’ average height and weight changes overtime, and to see if there is any correlation between player’s size and performance, finally try to find out the range of ideal weight and height for each of the five different player positions.
